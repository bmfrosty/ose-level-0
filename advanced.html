OSE Advanced Character Generator (Levels 1+)

🚧 Coming Soon!

This page will allow you to generate Advanced OSE characters for levels 1-14.

Advanced Mode separates race from class, allowing combinations like Dwarf Fighter, Elf Magic-User, etc.

Planned Features:

  • Level Selection: Choose character level (1-14)
  • Race Selection: Human, Dwarf, Elf, Gnome, Halfling
  • Class Selection: Cleric, Fighter, Magic-User, Thief (+ Spellblade in Smoothified Mode)
  • Racial Abilities: Infravision, detect doors, saving throw bonuses, etc.
  • Ability Score Adjustments: Racial modifiers (e.g., Dwarf +1 CON, -1 CHA)
  • Ability Score Generation: Roll 3d6 for each ability, apply racial adjustments
  • Class Requirements: Enforce minimum ability scores
  • Hit Points: Roll appropriate hit dice for class and level
  • Saving Throws: Use class-based progression tables + racial bonuses
  • Attack Bonus: Use class-based progression tables
  • Spell Slots: For Clerics, Magic-Users, and Spellblades
  • Thief Skills: For Thieves
  • Class Abilities: Display abilities gained at each level
  • XP Tracking: Show XP required for current and next level
  • Equipment: Starting equipment based on class
  • Export Options: PDF, PNG, Markdown, JSON

🏰 Smoothified Mode Support:

Advanced mode includes support for The Ruins of Castle Gygar campaign setting:

  • Modified Progressions: Different saving throw and attack bonus tables
  • Spellblade Class: Unique fighter/magic-user hybrid class
  • Elves as Spellblades: Elves use Spellblade progression in Smoothified Mode
  • Smoothed Progressions: More gradual advancement curves
