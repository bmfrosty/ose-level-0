# OSE Standard vs Gygar Mode Comparison

This document compares the progression differences between OSE Standard and Gygar Mode classes.

## Overview

Gygar Mode takes the OSE Standard classes and applies consistent smoothing to their progression tables, particularly for to-hit bonuses and saving throws. The goal is to provide more frequent, incremental improvements rather than large jumps at specific levels.

## Key Philosophy

**OSE Standard:** Uses irregular progression with large jumps at specific breakpoints (typically levels 4, 7, 10, 13).

**Gygar Mode:** Smooths progressions to provide more consistent, gradual improvements nearly every level.

## To-Hit Progression Categories

Both OSE Standard and Gygar Mode use three distinct to-hit progression tracks, carefully assigned to classes based on their combat role. The key difference is that OSE uses irregular jumps while Gygar smooths them into gradual increases.

**Fighter Progression** (Best combat capability):
- **OSE:** Used by Fighter, Dwarf, Elf, Halfling - irregular jumps at levels 4, 7, 10, 13
- **Gygar:** Used by Fighter, Dwarf, Elf, Halfling, Spellblade - smooth progression
- **Gygar Progression:** +1, +2, +3, +4, +5, +5, +6, +6, +7, +7, +7, +8, +9, +9
- **Design:** Intentionally front-loaded so fighters are better at hitting things starting at level 1

**Cleric/Thief Progression** (Moderate combat capability):
- **OSE:** Used by Cleric, Thief - irregular jumps at levels 5, 9, 13
- **Gygar:** Used by Cleric, Thief - smooth progression
- **Gygar Progression:** +0, +1, +1, +2, +2, +3, +3, +4, +5, +5, +6, +6, +7, +7
- **Design:** Starts slower than fighters but maintains steady growth

**Magic-User Progression** (Weakest combat capability):
- **OSE:** Used by Magic-User, Gnome - irregular jumps at levels 6, 11
- **Gygar:** Used by Magic-User, Gnome - smooth progression
- **Gygar Progression:** +0, +0, +1, +1, +2, +2, +2, +3, +3, +4, +4, +5, +5, +5
- **Design:** Slowest progression, as these classes rely primarily on magic

## Fighter Comparison

### To-Hit Progression

**OSE Standard:**
- Levels 1-3: 19 [0] (no bonus)
- Levels 4-6: 17 [+2] (jump of +2)
- Levels 7-9: 14 [+5] (jump of +3)
- Levels 10-12: 12 [+7] (jump of +2)
- Levels 13-14: 10 [+9] (jump of +2)

**Gygar Mode:**
- Linear progression: +1, +2, +3, +4, +5, +5, +6, +6, +7, +7, +7, +8, +9, +9
- Increases nearly every level or every other level
- Same final value at level 14: +9

### Saving Throws

**OSE Standard:** Large jumps at levels 4, 7, 10, 13

**Gygar Mode:** Gradual improvements nearly every level, reaching same or similar final values

**Result:** Mechanically equivalent at max level, but Gygar provides smoother power curve.

## Demihuman Class Feature Differences

### Requirements

**OSE Standard:**
- Demihumans are race-as-class (e.g., "Dwarf" is both race and class)
- Requirements are racial (e.g., "Minimum CON 9" for Dwarf)

**Gygar Mode:**
- Demihumans can choose classes (e.g., a Dwarf can be a Cleric, Fighter, etc.)
- Requirements specify race (e.g., "Dwarf. Minimum CON 9")
- Allows for more flexible character creation

### XP Bonuses

**OSE Elf:**
- +5% XP bonus if both INT and STR are 13+
- +10% XP bonus if INT 16+ and STR 13+

**Gygar Elf:**
- +5% XP bonus if both INT and STR are 13+
- **No +10% bonus option** (simplified)

**OSE Halfling:**
- +5% XP bonus if one prime requisite is 13+
- +10% XP bonus if both DEX and STR are 13+

**Gygar Halfling:**
- +5% XP bonus if one prime requisite is 13+
- **No +10% bonus option** (simplified)

**OSE Gnome:**
- +5% XP bonus if both DEX and INT are 13+
- +10% XP bonus if DEX 13+ and INT 16+

**Gygar Gnome:**
- +5% XP bonus if both DEX and INT are 13+
- **No +10% bonus option** (simplified)

### Class Availability

**OSE Standard:**
- Cleric: Human only
- Dwarf: Race-as-class only
- Elf: Race-as-class only
- Halfling: Race-as-class only
- Gnome: Race-as-class only

**Gygar Mode:**
- Cleric: Human, Dwarf, or Gnome (Dwarf/Gnome require WIS 9+)
- Dwarf: Can be any class appropriate to race
- Elf: Can be any class OR use Elf race-as-class OR Spellblade
- Halfling: Can be any class appropriate to race
- Gnome: Can be any class OR use Gnome race-as-class

### Racial Abilities

**All racial abilities remain identical between OSE and Gygar:**
- Dwarf: Detect construction tricks, detect room traps, infravision 60', listen at doors
- Elf: Detect secret doors, ghoul immunity, infravision 60', listen at doors
- Halfling: Defensive bonus vs large, hiding, initiative bonus, listen at doors, missile bonus
- Gnome: Defensive bonus vs large, detect construction tricks, hiding, infravision 90', listen at doors, speak with burrowing mammals

**Key Difference:** In Gygar Mode, these are racial abilities that apply regardless of class choice, not class features.

## Summary of Changes

### What Stays the Same

- **Hit Dice:** Identical
- **XP Requirements:** Identical
- **Spell Progression:** Identical (for spellcasting classes)
- **Racial Abilities:** Identical (detect doors, hiding, infravision, etc.)
- **Final Values:** Same or very similar at maximum level

### What Changes

- **To-Hit Progression:** Smoothed from large jumps to gradual increases
- **Saving Throws:** More frequent improvements instead of flat periods
- **Power Curve:** More consistent growth throughout levels

### Equivalent But Different

- **OSE:** Sudden power spikes at specific levels
- **Gygar:** Steady, incremental power growth
- **Result:** Same destination, different journey

## Design Philosophy

**OSE Standard** follows the classic B/X D&D design where:
- Characters experience distinct "tiers" of power with clear breakpoints
- Demihumans are race-as-class (your race determines your class)
- Higher XP bonuses available for exceptional ability scores

**Gygar Mode** maintains the same overall power level but:
- Distributes improvements more evenly, reducing "dead levels"
- Separates race from class (demihumans can choose classes)
- Simplifies XP bonuses (max +5% instead of +10%)
- Provides more frequent rewards for leveling up

Both approaches are valid and create slightly different play experiences:
- **OSE:** Emphasizes milestone moments, clear power tiers, and race-as-class tradition
- **Gygar:** Emphasizes consistent growth, flexible character creation, and frequent small victories

The choice between them is primarily aesthetic and depends on preferred play style and campaign needs.
