OSE Basic Character Generator (Levels 1+)

🚧 Coming Soon!

This page will allow you to generate Basic OSE characters for levels 1-14.

Basic Mode uses the traditional OSE race-as-class system where Dwarf, Elf, and Halfling are classes (not races).

Planned Features:

  • Level Selection: Choose character level (1-14)
  • Class Selection: Cleric, Fighter, Magic-User, Thief, Dwarf, Elf, Halfling
  • Ability Score Generation: Roll 3d6 for each ability
  • Class Requirements: Enforce minimum ability scores
  • Hit Points: Roll appropriate hit dice for class and level
  • Saving Throws: Use class-based progression tables
  • Attack Bonus: Use class-based progression tables
  • Spell Slots: For Clerics, Magic-Users, and Elves
  • Thief Skills: For Thieves
  • Class Abilities: Display abilities gained at each level
  • XP Tracking: Show XP required for current and next level
  • Equipment: Starting equipment based on class
  • Export Options: PDF, PNG, Markdown, JSON
