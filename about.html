About These Tools

📖 Project Background

These character generators and class references were created by the author to support running a Ruins of Castle Gygar campaign using the Old-School Essentials (OSE) ruleset.

The tools include both the standard OSE classes and "smoothified" versions with more gradual progression curves, as well as the Spellblade class—a fighter/magic-user hybrid designed specifically for the Gygar campaign setting.

🏰 Campaign Evolution

The author initially started the campaign using Basic OSE (race-as-class system), where Dwarf, Elf, and Halfling are classes rather than races. After a few sessions, the campaign transitioned to Advanced OSE, which separates race from class and allows for more flexible character combinations.

Because of this mid-campaign transition, these tools support both approaches:

  • Basic Mode: Traditional race-as-class system
  • Advanced Mode: Separate race and class selection
  • OSE Standard: Original progression tables with irregular jumps
  • Gygar Mode: Smoothed progressions with more frequent, incremental improvements

✨ Key Features

  • Level 0 Character Generator: Create 0-level characters for funnel-style adventures
  • Complete Class Documentation: All OSE classes with full progression tables
  • Smoothified Progressions: Gygar Mode variants with gradual advancement
  • Spellblade Class: Unique warrior-mage hybrid for Gygar campaigns
  • Comparison Tools: Side-by-side analysis of OSE vs Gygar progressions
  • Both Basic and Advanced: Support for race-as-class and race+class systems
