🏰 Campaign Background

These character generators were created to support running a campaign of Ruins of Castle Gygar using the Old-School Essentials (OSE) ruleset.

The author initially started the campaign using Basic OSE (race-as-class system), where Dwarf, Elf, and Halfling are classes rather than races. After a few sessions, the campaign transitioned to Advanced OSE, which separates race from class and allows for more flexible character combinations.

The "Smoothified Mode" progressions were designed to provide more gradual advancement and remove demihuman level limits, making the game feel more balanced and rewarding for players who choose non-human characters.

✨ Key Features

  • Two Modes in One Generator: Basic (race-as-class, Levels 0–14) and Advanced (race + class, Levels 0–14) — switch between them at any time
  • Three Progression Modes: OSE Standard, Smoothified, and Labyrinth Lord
  • Demihuman Level Limits: Optional in Basic Mode (standard or extended to 14), configurable in Advanced Mode
  • Complete Class Documentation: All OSE classes with full progression tables
  • Spellblade Class: Warrior-mage hybrid class option
  • Flexible Options: Ability score minimums, HP requirements, level 0 HP inclusion, prime requisite modes
  • Export Options: PNG and PDF via print dialog on all character sheets
